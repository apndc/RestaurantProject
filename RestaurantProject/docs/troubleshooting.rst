Troubleshooting
===============
This section describes some common issues that can arise and possible solutions.

Issue #1
--------
Description:
Cannot Connect to PostgreSQL Database

Solution:
Make Sure PostgreSQL Service is Running and All Fields Needed in .env File In App Route are Correctly Set

Issue #2
--------
Description:
Outdated requirements.txt Causing Problems

Solution:
Manually doing a pip install <dependancy-name> for Each Missing Dependancy

Issue #3
--------
Description:
Application Crashes on Startup

Solution:
Check the Application Logs for Error Messages and Tracebacks to Identify the Root Cause.
